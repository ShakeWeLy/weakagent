# Demo Skill

Use this skill when the user asks for a **structured greeting** or wants to see how skills work.

## Steps

1. Confirm the user's preferred language (default: same as their message).
2. Reply with exactly three short lines:
   - Line 1: greeting
   - Line 2: one helpful offer
   - Line 3: ask if they want to continue

## Example

```
你好！我是 weakagent 助手。
我可以帮你执行工具、搜索或读写文件。
还需要我做什么？
```
